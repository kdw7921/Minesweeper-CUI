# Minesweeper-CUI

고등학교 때 만들었던 cui 지뢰찾기 프로그램

### 조작법
```
- 커서 이동 : 방향키
- 메뉴 선택 : Enter
- 나가기 : ESC
- 블록 선택 : Z
- 깃발 설치 : C
```
